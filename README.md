# Main-Campus-Branch-Campus-Network

## 📌 Project Overview
This project is a complete enterprise-level campus network designed using Cisco Packet Tracer.  
The network consists of a Main Campus and a Branch Campus connected through routers and Layer 3 switching.

The project demonstrates:
- VLAN segmentation
- Inter-VLAN routing
- Layer 2 & Layer 3 switching
- Department-wise network design
- Server integration
- Branch-to-main campus communication

---

# 🏢 Network Architecture

## Main Campus Network
The Main Campus contains multiple departments distributed across different buildings:

### Building A
- Admin Department → VLAN 10 → 192.168.1.0/24
- HR Department → VLAN 20 → 192.168.2.0/24
- Finance Department → VLAN 30 → 192.168.3.0/24
- Business Department → VLAN 40 → 192.168.4.0/24

### Building B
- Engineering & Computing → VLAN 50 → 192.168.5.0/24
- Arts & Design → VLAN 60 → 192.168.6.0/24

### Building C
- Student Labs → VLAN 70 → 192.168.7.0/24
- IT Department → VLAN 80 → 192.168.8.0/24

### Branch Campus
- Staff Department → VLAN 90 → 192.168.9.0/24
- Student Lab → VLAN 100 → 192.168.10.0/24
